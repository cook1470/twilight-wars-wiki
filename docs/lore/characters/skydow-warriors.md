# 天影十字軍 人物誌

## 關鍵人物

- [影娘](</lore/characters/details/影娘.md>) - 天影十字軍薔薇刺客，負責主持學員的覺醒試煉。
- [影俠](</lore/characters/details/影俠.md>) - 天影精英，本名隱奘，師承天隱門。秘藥事件失蹤後與木舍利結合，成為七月教教主。
- [隱知墨](</lore/characters/details/隱知墨.md>) - 天隱門傳人，影俠與隱心的同門師弟。
- [隱心](</lore/characters/details/隱心.md>) - 天隱門弟子，隱知墨之妹，影俠的青梅竹馬。
- [七月教護法](</lore/characters/details/七月教護法.md>) - 七月教的高階侍從與護衛。
- [七月教教主](</lore/characters/details/七月教教主.md>) - 七月教最高領導者，實為馬茲元精掌控的影俠肉身。
- [教主](</lore/characters/details/教主.md>) - 同「七月教教主」，為馬茲元精掌控的影俠肉身。
- [影天真](</lore/characters/details/影天真.md>) - 與影俠相關的天影戰士。
- [初影挑戰者](</lore/characters/details/初影挑戰者.md>) - 天影基層武士，初影破頂修羅的挑戰對象。
