# 阿薩斯

阿薩斯城的領袖，暗中瞞著帝國進行上古「冰劍」的祕密研究。

## 背景資訊

### 地位與權力
阿薩斯 是哈斯卡帝國邊境重鎮「阿薩斯城」的領袖。他在名義上效忠於哈斯卡大帝並受其管轄，同時指揮著當地的阿薩斯守衛隊與騎士團。
