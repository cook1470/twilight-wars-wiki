# 伊拉馬人質

被困於迷霧森林中的伊拉馬部落成員，不具備武裝。

## 背景資訊

被困於迷霧森林中的伊拉馬部落女性成員。

在任務「迷霧森林」初期，她被盜礦賊擄走作為人質。此時她頭上顯示的名稱為「伊拉馬人質」。當玩家（天影武士）成功將其救出後，會得知她的真實姓名為「伊芙」，且其名稱標籤也會隨之變更。她與長老伊拉馬關係密切，是部落中的核心成員。
