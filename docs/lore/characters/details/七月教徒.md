# 七月教徒

第七月教團的門徒，多為前傭兵成員。

## 背景資訊

駐守於赤日祭壇四周的武裝力量。他們原為馬茲席塔的部下，在馬茲席塔建立教團後轉為教徒，負責守護祭壇並監視教主的動向。
