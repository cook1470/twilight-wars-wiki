# 小哈片刻

Gamelet.Online 創辦人，光暈世界觀的「創世神」與哈斯卡大帝的現實原身。

## 背景資訊

嘎姆擂台（Gamelet.Online）創辦人，其使用者 ID 為 「haska」。

### 角色考據與開發紀錄

小哈片刻是嘎姆擂台平台的創始成員與核心工程師。在《光暈戰記》的世界觀中，哈斯卡帝國（Haska Empire）的名稱與其使用者 ID 「haska」直接對應。

其化身「哈斯卡大帝」在遊戲劇情中具備不老不死的超然特質。在早期開發階段，他與共同開發者「藍天白雲（hychen）」共同負責平台的維護與多款遊戲的發布。
