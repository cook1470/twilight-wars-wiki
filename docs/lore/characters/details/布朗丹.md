# 布朗丹

銀邊盜賊團首領。

## 背景資訊

布朗丹，人稱「銀邊」，是活躍於波姆峽谷一帶的銀邊盜賊團首領。他在波姆峽谷兩岸險峻的山勢中設下伏擊，企圖打劫過往的旅人。在皇家騎士團護送阿薩斯危城急報穿越峽谷時，布朗丹率領銀邊盜賊團與皇家軍隊展開了激戰。
