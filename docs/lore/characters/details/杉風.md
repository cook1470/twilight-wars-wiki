# 杉風

黑森盜賊團幹部，首領副手。

## 背景資訊

黑森盜賊團幹部。

德里克．黑森的副手。在任務「決戰黑森盜賊團」中與首領一同出現並參與戰鬥。配備盜賊軍帽。主要使用霰彈槍與小刀。

備註：其姓名與另一位副手「典火」具備「搧風點火」之諧音趣味。
