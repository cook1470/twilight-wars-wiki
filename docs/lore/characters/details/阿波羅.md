# 阿波羅

哈斯卡大帝之子，皇家第一騎士，統帥皇家騎士團的領袖，與九方一色是生死之交。

## 背景資訊

全名：阿波羅．葛羅因艾斯．伏朗．佛依得斯特．哈斯卡

### 身分與地位
阿波羅是哈斯卡大帝的雙生子，雅堤米絲的親弟。作為皇家騎士團的第一騎士與最高統帥，他繼承了不老不死的皇家血脈，並以強大的戰鬥力與領袖魅力守護著帝國的尊嚴。
