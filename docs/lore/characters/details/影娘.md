# 影娘

天影十字軍薔薇刺客，負責主持學員的覺醒試煉。

## 背景資訊

影娘是引導天影武士踏上修行之路的關鍵導師。她在《初影破頂修羅》系列中擔任修業的主持者。在最重要的覺醒儀式中，她透過決鬥將體內的「初影之氣」灌通入玩家的天靈要穴；只有在這種生死邊緣的磨礪下，學員才能真正打開體內潛藏的天影原力。她是玩家從「凡人」蛻變為「真影」的見證者與引路人。
