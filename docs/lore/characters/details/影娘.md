# 影娘

天影十字軍薔薇刺客，負責主持學員的覺醒試煉。

## 背景資訊

影娘是天影十字軍的薔薇刺客，也是指引天影武士修業的師父。她在《初影之源》任務開場負責介紹決鬥規則，確保勝出者的天靈要穴能貫通「初影之氣」以打開天影力量。該任務在舊版中限定雙人遊玩（由兩名玩家決鬥），新版則支援單人對抗遊戲 AI。她在《初影破頂修羅》章節的前三場任務中均有出現。
