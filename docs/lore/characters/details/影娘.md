# 影娘

天影十字軍薔薇刺客，負責主持學員的覺醒試煉。

## 背景資訊

影娘是引導天影武士踏上修行之路的關鍵導師。她在《初影破頂修羅》系列中擔任修業的主持者。在《初影之源》的覺醒儀式中，她負責指引學員進行決鬥，確保「初影之氣」能貫通存活者的天靈要穴；只有在這種考驗下，學員才能真正打開體內潛藏的天影原力。

在早期的修煉機制中，這場決鬥通常需要兩名學員（玩家）互相競爭，僅有勝者能晉級；隨著時代演進，目前的修行已支援單人進行，由影娘監督學員與具備人工智慧的對手進行決鬥。影娘始終作為天影十字軍的見證者，監控著學員從「初影」蛻變為「真影」的完整修煉過程。
