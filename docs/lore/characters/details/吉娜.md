# 吉娜

傭兵團八手，受制於馬茲席塔的神秘女性。

## 背景資訊

傭兵團成員，位列八手。

在「噬魂秘藥」中持有能解除屍毒的「回魂解藥」，是天影武士追擊的目標。在隨後的「黑色追殺令」與「斷魂殘影」等多個章節中均有出現。在「斷魂殘影」事件中，她與隱心一同參與了關於隱奘的斷魂之儀。

她是青嵩散人（隱居士）的弟子，並在隱居士的指示下，作為內應待在馬茲席塔（元精）身邊。在「盜體合靈」事件中，她將希瓦暗道的情報傳遞給主角，是推動計畫的關鍵人物。

備註：馬茲席塔手中握有她的精魂氣魄，因此她表面上對其極為恭順。其複雜的雙重身份使其在多個陣營衝突中扮演了關鍵角色。
