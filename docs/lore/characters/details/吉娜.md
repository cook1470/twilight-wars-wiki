# 吉娜

傭兵團八手，情報販子，青嵩散人之徒，曾作為內應潛伏於馬茲席塔身邊。

## 背景資訊

### 身分與職位

吉娜 是傭兵團中排名第八的高手，對外以「情報販子」身分活動，並曾自誇為房地產仲介，展現其廣闊的情報網。實際上，她是青嵩散人（隱居士）的親傳弟子，受師命潛伏在傭兵團七手 馬茲席塔 身邊擔任內應。
