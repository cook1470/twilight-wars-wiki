# 馬茲席塔元精

馬茲席塔三元神之一，繼承其黑暗面。

## 背景資訊

馬茲席塔三元神之一，主地，繼承了內心的黑暗面。

在元體戰敗後從地城復出，重新奪回七手之位並找尋元體舍利。他習得神魂分離之術，在「希瓦煉獄」中奪取瀕死者的魂魄以壯大勢力。他後來挾持影俠的肉身創立「七月教」。最終在「浴血送魂曲」中試圖強行與影俠結合時，中下元靈的計策而被消滅。
