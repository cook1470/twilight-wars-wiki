# 隱居士

前傭兵團五手，能穿梭表、裏兩界的悟道者。

## 背景資訊

前傭兵團五手，光暈戰記世界觀中的核心人物之一。

他曾與九方織夢共同研習「腦洞」之學，但在九方織夢走火入魔並陷入長眠後，他將其元神、元體與元靈分離，隨後在與九方一色的戰鬥中撤退，隱居於怒江上游。

十年後，他在「黑色追殺令」中救下被馬茲席塔追殺的主角，並引導其尋找吉娜。隨後在「盜體合靈」與「斷魂殘影」事件中多次現身協助主角。他的元靈則被封印於風月寶鏡中，在「龍魅奇謀」中以「蕭居士」的身分指引天影武士進入腦洞斬殺火龍。

他是極少數能穿梭表、裏兩界的悟道者，也是「九龍大牢」等強大術法的施展者。其身分橫跨多個陣營與時代，是揭開世界觀真相的關鍵。
