# 九方一色

黑炎大將軍，傭兵團五手。

## 背景資訊

九方一色曾是北方寒地君主，為了尋找女兒九方織夢加入傭兵團並接下五手位置。他持有黑炎劍，與阿波羅是生死之交。曾在赤焰山擊退青嵩散人。
