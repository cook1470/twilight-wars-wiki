# 伊拉馬

伊拉馬部落的長老，語言簡練，委託天影武士救回族人。

## 背景資訊

伊拉馬部落的長老。

居住在「迷霧森林」北方的部落領地。與外界溝通時使用極其簡練、破碎的語言（如：「天影。救人。」）。

在「迷霧森林」事件中，他會將部落的武器「水晶弓」贈與展現出實力的天影武士，並委託其前往東南方的盜賊營地救回被擄走的族人。他是該任務的關鍵引導者與目標達成點。
