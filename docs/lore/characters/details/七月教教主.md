# 七月教教主

七月教最高領導者，實為馬茲元精掌控的影俠肉身。

## 背景資訊

七月教教主自稱為「影俠」，駐守於紅日聖壇的秘室。因其肉身已與「木之聖遺」（木舍利）結合，故具備不死之身與無限增生之力。

該身軀實質上受到馬茲席塔「元精」的操控，作為吸引天影高手與傭兵投靠的象徵。在《俠影之亂》章節中，天影高手的任務目標即為進入聖壇核心找出七月教教主，以查明其與失蹤影俠的關係。

在不同劇本中（如《忘情谷》），該角色常被簡稱為「教主」。
