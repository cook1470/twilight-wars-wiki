# 教主

同「七月教教主」，為馬茲元精掌控的影俠肉身。

## 背景資訊

「教主」是開發者在劇本簡化時對「七月教教主」的稱呼。兩者為同一角色實體，代表與木舍利結合、受馬茲席塔操控的影俠肉身。
