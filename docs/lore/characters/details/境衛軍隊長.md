# 境衛軍隊長

駐守北方邊境地殼裂縫的帝國軍官。

## 背景資訊

隸屬於亞拜爾上校的軍官。在任務「穿越北方邊境」中負責帶領士兵在地殼裂縫一帶執行邊境警戒與防衛任務。
