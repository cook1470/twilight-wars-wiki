# 伊蒂絲

地球復原組織基因工程權威，即阿曼達博士。

## 背景資訊

伊蒂絲是地球復原組織 (EBO) 的阿曼達博士。曾被馬茲席塔綁架以奪取其關於屍血蟲與古代生物研究的關鍵數據。若主角選擇救她則會開啟【情份】任務。
