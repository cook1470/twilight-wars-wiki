# 影俠精魂

影俠（隱奘）的靈魂型態。

## 背景資訊

在《浴血送魂曲》中，透過馬茲席塔施展的「裂魂之儀」，隱奘的意志被從與木舍利結合的肉身中剝離，轉化為精魂型態。此過程旨在切斷他與木之聖遺的血脈連結，使靈核之主能吸收聖遺，並讓隱奘找回失落的記憶。
