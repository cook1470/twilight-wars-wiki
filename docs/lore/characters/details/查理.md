# 查理

皇家科學院研究員，駐守於阿薩斯城，協助進行上古冰劍的能量測試。

## 背景資訊

查理 是隸屬於哈斯卡帝國「皇家科學院」的研究員，被派駐在阿薩斯城協助 阿薩斯 進行研究。在《冰劍檔案：冰原遺跡》任務中，他隨同調查隊深入極地廢墟，負責操作儀器進行「上古冰劍」的能量共鳴測試。
