# 第五疑鬼

修行者內心的猜忌與懷疑具現化。

## 背景資訊

風月寶鏡試煉中的六毒心魔之一。在《風月寶鏡》章節的「古鏡重明」任務中，於神光目眩之際出現，是修行者必須克服的內心障礙。演員外表會隨著玩家所選的角色而改變。
