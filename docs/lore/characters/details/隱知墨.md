# 隱知墨

天隱門傳人，影俠與隱心的同門師弟。

## 背景資訊

出身天隱門，師承眉隱大師。年幼時與妹妹隱心失散並陷入戰亂，後被少年隱奘（影俠）救出並引薦入同門。與隱奘、隱心為形影不離的摯友。在隱奘入世加入天影並失蹤後，他閉關修煉七年有成。出關後發現摯友失蹤、妹妹亦踏入江湖，遂展開尋人旅程。他與玩家並肩作戰，調查七月教真相，並前往丹格斯城風月寶塔尋求古鏡之力，試圖揭開七月教主的真面目。
