# 腦洞主

青嵩散人元靈初登場時的稱呼。

## 背景資訊

青嵩散人（隱居士）的元靈，在「龍魅奇謀」章節中初次登場時的稱呼。

在任務「尋龍訣」中，玩家與隱知墨走散後被捲入腦洞旋渦，遇到了這名自稱為「腦洞之主」的神祕人物。他隨後揭露自己舊姓蕭，並要求玩家稱呼其為「蕭居士」。他向玩家解釋了腦洞的奧祕，並指引玩家前往異界救人。他在劇情中起到了世界觀解說與關鍵引導的作用。
