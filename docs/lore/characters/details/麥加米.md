# 麥加米

浣熊村村民，秘藥實驗的倖存者。

## 背景資訊

居住在紅河上游浣熊村的村民。

自稱為秘藥實驗的對象，在「活體實驗村」事件中帶領天影武士前往他的家鄉調查真相。他在劇本中扮演了引導者與保護對象的角色，其生存與否直接關係到任務的成敗。
