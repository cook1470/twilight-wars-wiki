# 麥加米

自稱為秘藥實驗對象的男人，來自浣熊村。

## 背景資訊

麥加米 (Mykami) 是一位原本居住在紅河上游浣熊村的村民。他聲稱自己是傭兵團「噬魂秘藥」活體實驗的受害者，並引導天影十字軍的主角前往已被傭兵團佔領的浣熊村進行調查。他在《噬魂秘藥》系列任務中是引發劇情的重要人物。
