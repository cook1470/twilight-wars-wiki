# 影天真

與影俠相關的天影戰士。

## 背景資訊

在《斷魂殘影》章節中登場。最初以天影武士身分出現在「聖壇末日」的劇場中，隨後在「忘情谷」中以第三勢力身分現身。其具體背景與影俠（隱奘）有著密切的關聯，是該時期重要的劇場角色。
