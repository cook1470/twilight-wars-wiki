# 青嵩散人

前傭兵團五手，隱居士的前身。

## 背景資訊

前傭兵團五手。

與九方織夢共同研習「腦洞」之學的高人。在織夢走火入魔後，曾於赤焰山與前來尋女的九方一色激烈交戰，隨後撤退並隱居於怒江上游，更名為「隱居士」。他是揭開腦洞奧祕的先驅者之一。
