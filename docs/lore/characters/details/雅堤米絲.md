# 雅堤米絲

哈斯卡大帝之長女，靈能研究所主席，創造了「光印」技能。

## 背景資訊

全名：雅堤米絲．克莉絲托爾．歐芙．妲可妮茲．哈斯卡

### 身分與地位

雅堤米絲 是哈斯卡大帝的雙生長女，阿波羅的親姊。她在帝國內部繼承了不老不死的皇家血脈，並擔任「靈能研究所」主席。
