# 馬茲席塔元靈

馬茲席塔三元神之一，收玩家為徒的導師。

## 背景資訊

馬茲席塔三元神之一，主天，沉靜如空。

曾被元體關押在希瓦地城最深處。在「地宮元神」中被主角發現，隨後收其為徒並傳授「破魂血刃」絕技。他是馬茲席塔三元歸一的核心意志。在「斷魂殘影」中與影俠二體合一，最終主導斷魂之儀使馬茲席塔徹底復活重生。
