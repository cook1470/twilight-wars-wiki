# 無天礦賊團大頭目

無天礦賊團首領，在火山口一帶進行盜礦活動。

## 背景資訊

無天礦賊團的首領。

在任務「天火的試煉」中率領成員在聖塔那古火山口附近活動。他們是為了挖掘火山口附近的稀有礦石而冒險深入禁區的非法組織。主要使用霰彈槍與小刀。
