# 負傷的盜賊

黑森盜賊團成員，在「跟入敵窟」中被玩家跟蹤。

## 背景資訊

黑森盜賊團成員。

在任務「跟入敵窟」中，於「保衛艾爾瑪」戰鬥結束後企圖逃走。玩家需對其進行跟蹤以發現盜賊團總部的入口。他在劇情中起到了引導玩家發現賊窟位置的作用。
