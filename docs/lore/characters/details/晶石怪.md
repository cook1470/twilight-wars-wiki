# 晶石怪

守護火焰晶石、體內含有高能量的魔物。

## 背景資訊

存在於聖塔那古火山口附近山洞中的魔物。

在任務「火焰晶石的秘密」中出現。與一般的石怪不同，它們體內含有高純度的晶石能量，會散發出暗淡的紅光。它們守護著解開地火靈魂傳說關鍵的火焰晶石。雖然在論壇統計中被標註為帝國陣營，但其本質更接近於環境中自發生成的元素生物（第三勢力）。
