# 蕭居士

青嵩散人的元靈，受困於風月寶鏡十年。

## 背景資訊

青嵩散人的「元靈」。

十年前被封印於風月寶鏡之中。在「龍魅奇謀」事件中，將進入密室的天影武士與隱知墨吸入腦洞世界。他指引玩家尋找龍穴，並施展「九龍大牢」輔助玩家斬殺火龍。在火龍被滅後，因消耗過度而陷入昏厥。
