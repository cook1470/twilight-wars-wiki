# 盜礦賊頭目

非法盜礦組織的地區領導者。

## 背景資訊

非法盜礦組織的地區領導者。

負責指揮盜礦賊在北方邊境禁區內的挖掘活動。在任務「穿越北方邊境」與「天火的試煉」中均有出現。相較於一般的盜礦賊，他們具備更強的武裝與組織能力，主要使用大刀或霰彈槍。在劇情中，他們是為了守護非法所得的礦石而與玩家發生衝突。
