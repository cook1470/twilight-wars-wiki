# 隱心

天隱門弟子，隱知墨之妹，影俠的青梅竹馬。

## 背景資訊

出身天隱門，師承眉隱大師。幼時與哥哥隱知墨在戰亂中被少年隱奘（影俠）救出並引薦入同門，三人為青梅竹馬。在影俠失蹤後，她離開閉關中的哥哥獨自踏入江湖尋訪。為了查明真相，她化名加入「第七月教團」擔任教主護法（侍從）。在《斷魂殘影》事件中，她與玩家及馬茲席塔元靈合作，試圖透過儀式找回影俠失去的記憶。
