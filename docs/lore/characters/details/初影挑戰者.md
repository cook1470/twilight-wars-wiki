# 初影挑戰者

天影基層武士，初影破頂修羅的挑戰對象。

## 背景資訊

在「初影破頂修羅」章節的「初影之源」任務中登場。該任務的核心機制是新手武士間的對決，旨在貫通「初影之氣」。若玩家選擇單人遊玩，系統會生成「初影挑戰者」作為對手；若為雙人遊玩，則該職位由另一位玩家擔任，此實體不會出現。其演員外表為革命鬥士，使用手槍與小刀進行戰鬥。
