# 馬茲席塔元體

馬茲席塔三元神之一，代表其力量與霸道的設定身分。

## 背景資訊

馬茲席塔三元神之一，代表力量與霸道。

在「黑色追殺令」章節中擔任傭兵團七手。他在三人出關之際，將元精與元靈鎖入希瓦地城。最終在「勝者為王」任務中被主角擊敗，其精神意志隨之潰散，體內的元素靈子形成舍利子四散各處。
