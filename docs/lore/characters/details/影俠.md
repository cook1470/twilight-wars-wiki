# 影俠

天影精英，本名隱奘，師承天隱門。秘藥事件失蹤後與木舍利結合，成為七月教教主。

## 背景資訊

### 早年與天隱門

影俠本名**隱奘**，出身於行事低調、追求仙道的「天隱門」，師承**眉隱大師**。少年時期的隱奘曾從戰亂中救出**隱知墨**與**隱心**兩兄妹，並引薦他們拜入同門，三人成為青梅竹馬的摰友。由於隱奘生性好動、熱愛交友，與天隱門清靜無為的作風不合，他在「稜光地會」閉關僅一年便決定出關，捨棄本門正道，投身江湖並加入天影十字軍。

### 天影精英與秘藥事件

加入天影後，隱奘以「影俠」之名行走江湖，並成為天影議會在紅河地區的核心執行官。在「秘藥事件」期間，他與玩家指揮官及天影精英部隊並肩作戰，調查屍毒起源。然而，在《秘藥之祕》中，影俠與吉娜進行決戰後雙雙失蹤，自此行蹤成謎，民間甚至出現了拜祭「影俠尊者」的廟宇。

### 七月教主與木之聖遺

失蹤十年後，影俠以「七月教主」的身分重新現世。此時他的身軀已與**木之聖遺（木舍利）**結合，具備不死之身與無限增生之力。然而，這具身軀實際受到**馬茲席塔**的掌控，做為吸引天影高手與傭兵投靠的旗幟。雖然隱奘表面上對過去毫無所知，但其內心深處並無迷惘，他深信自己乃天命所歸，欲藉由體內的力量進入虛無之境，窺見宇宙真理。在《浴血送魂曲》中，馬茲席塔施展「裂魂之儀」，試圖將隱奘的意志從其與木舍利結合的肉身中剝離。
