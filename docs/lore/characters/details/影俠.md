# 影俠

天影武士精英，秘藥事件後失蹤，後被確認為七月教教主。

## 背景資訊

天影十字軍的高階成員。又名「隱奘」，是天影議會在紅河地區的核心執行官。在調查屍毒起源與尋找血清的過程中與玩家並肩作戰。其身軀後來與木之聖遺結合。
