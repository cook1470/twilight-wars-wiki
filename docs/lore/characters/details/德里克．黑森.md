# 德里克．黑森

黑森盜賊團的首領，率部騷擾艾爾瑪村落。

## 背景資訊

德里克．黑森 是黑森盜賊團的頭目。他帶領手下騷擾地處極北、原本未被帝國染指的日光地村落「艾爾瑪」，並將部分村民擄入賊窟。在任務《決戰黑森盜賊團》中，他帶著副手與手下從外歸來，隨後與前來解決事端的天影武士展開決戰。
