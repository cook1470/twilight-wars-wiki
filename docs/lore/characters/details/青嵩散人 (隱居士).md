# 青嵩散人 (隱居士)

前傭兵團五手，隱居於魔窟山脈。

## 背景資訊

青嵩散人（隱居士）是前傭兵團五手，提出了「腦洞」理論。曾與九方織夢一同對抗火龍，後導致元神、元體、元靈分離。他收傭兵主角為徒，並引導其對抗馬茲席塔。
