# 艾爾瑪村長

艾爾瑪村的村長。

## 背景資訊

艾爾瑪村長 是極北日光地村落艾爾瑪的領袖。在黑森盜賊團大舉騷擾村落並企圖擄走村民時，他帶領村民拿起簡陋武器進行抵抗，並向天影十字軍發出求援信號。
