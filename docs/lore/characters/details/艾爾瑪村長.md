# 艾爾瑪村長

艾爾瑪村的長老，掌握著關於「地火靈魂」的古老傳說。

## 背景資訊

艾爾瑪村的長老。

在天影武士尋找修行之路時，向其透露了關於艾爾瑪的傳說：『穿越天火的試練，讓地火的靈魂解脫，生命將得到溫度。』

在「艾爾瑪的紛亂」事件中，面對黑森盜賊團的侵擾，向抵達村落的天影武士請求協助，希望能保護村民免於盜賊的威脅。
