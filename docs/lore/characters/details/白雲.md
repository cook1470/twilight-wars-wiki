# 白雲

傭兵團成員，白雲護法。其名可能致敬現實開發者「藍天白雲」。

## 背景資訊

傭兵團成員，被稱為「白雲護法」。在《勝者為王》任務中，她與藍天一同守護著馬茲席塔。

其角色名稱與現實中嘎姆擂台共同開發者「藍天白雲（hychen）」的稱號完全重合。
