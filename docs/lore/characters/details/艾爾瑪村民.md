# 艾爾瑪村民

居住在極北之地艾爾瑪村的平民，在黑森盜賊團的侵擾中尋求天影武士的庇護。

## 背景資訊

生活在日光地極北村落「艾爾瑪」的村民。艾爾瑪地處偏遠，長年遠離戰爭，是個難得一見的世外桃源。\n\n然而隨著黑森盜賊團的興起，村民們的生活陷入危機。在「艾爾瑪的紛亂」事件中，部分村民被盜賊俘虜至賊窟作為人質，其餘留守村落的村民則與天影武士並肩作戰，守護家園。\n\n雖然不具備正規武裝，但他們的生存狀況直接關係到天影武士在該地區的任務成敗。
