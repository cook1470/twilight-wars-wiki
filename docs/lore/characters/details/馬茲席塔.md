# 馬茲席塔

傭兵團七手，一分為三的三元神主體。

## 背景資訊

馬茲席塔（馬茲）是傭兵團七手。因習得「分魂離魄之術」而分為元體、元精、元靈。元體被主角擊敗後化為五枚舍利，元精掌控精魂氣魄，元靈則收主角為徒。
