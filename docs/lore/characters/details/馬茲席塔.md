# 馬茲席塔

傭兵團七手，具備三位一體特質的神祕強者。

## 背景資訊

傭兵團七手，具備「三位一體」特質的核心人物。

數十年前透過古書悟得分魂離魄之術，分裂為「元體」、「元精」與「元靈」三個具備獨立意識的個體。

在「黑色追殺令」事件中，作為元體的他對主角下達追殺令，最終在「勝者為王」中被擊敗。隨後主角在「鬼城懺七」中發現其元神並未真正消亡，進而揭開了希瓦地城與精魂氣魄的祕密。他是推動傭兵線劇情的關鍵核心，也是「元素大法」與「破魂血刃」等強力術法的淵源。
