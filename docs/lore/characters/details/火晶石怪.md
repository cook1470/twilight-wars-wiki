# 火晶石怪

存在於「灼日龍炎」腦洞中的火屬性魔物。

## 背景資訊

出現在天影任務「灼日龍炎」中的火屬性魔物。

這些生物存在於一名女性靈劍使的「腦洞」（心靈空間）深處。由於該女子在練功時真氣走岔，其特殊的真氣能量從異界引來了強大的魔物，火晶石怪便是其中的組成部分。

在遊戲中，它們作為敵對生物出現，是心靈空間混亂象徵的具現化。由於其異界生物的本質，並不隸屬於日光地的任何常規陣營。
