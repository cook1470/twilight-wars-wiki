# 盜礦賊

在北方邊境禁區挖掘稀有礦物的非法成員。

## 背景資訊

非法組織的基層成員。

他們主要在北方邊境的地殼裂縫、迷霧森林以及聖塔那古火山口附近出沒。這些人冒險進入禁區是為了挖掘稀有的礦物或寶藏。在任務中，他們對所有闖入者（包括玩家）抱持敵意，是玩家在探索邊境初期經常需要面對的敵人。他們主要配備小刀、手槍或步槍。
