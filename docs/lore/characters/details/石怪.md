# 石怪

存在於火山口山洞中的土屬性魔物。

## 背景資訊

存在於聖塔那古火山口附近山洞中的土屬性魔物。

在任務「火焰晶石的秘密」中出現。它們由堅硬的岩石組成，與晶石怪一同守護著充滿能量的晶石洞穴。它們是地殼變動與地火能量結合後的產物，會主動攻擊闖入洞穴的生命體。
