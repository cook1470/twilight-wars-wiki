# 亞拜爾上校

皇家騎士團上校，駐守北方邊境的資深軍官，以忠誠與嚴謹著稱。

## 背景資訊

皇家騎士團的高階軍官，長期駐守在帝國北方邊境的大型地殼裂縫。

他是一名極度重視紀律與使命感的職業軍人。在「地火靈魂的傳說」事件中，他負責鎮守邊境關口，阻止任何武裝力量跨越邊境。他對帝國展現出絕對的忠誠，認為維持秩序高於一切。
