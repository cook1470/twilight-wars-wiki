# 亞拜爾上校

駐守於北方邊境地殼裂縫區域的帝國軍官，率領境衛軍。

## 背景資訊

### 職位與駐地
亞拜爾上校 是哈斯卡帝國的高級軍官，負責統轄駐守在北方邊境的帝國境衛軍。他的駐地位於艾爾瑪村北方的巨大地殼裂縫，該地因富含稀有礦物，除了帝國駐軍外，也有許多淘金族與盜賊出沒，是帝國嚴密監控的軍事防線。
