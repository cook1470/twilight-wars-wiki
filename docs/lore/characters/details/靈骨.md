# 靈骨

風月寶鏡內部的神祕守護者。

## 背景資訊

存在於天影神兵「風月寶鏡」內部的守護者。在《風月寶鏡》章節的最終任務「古鏡重明」中，當天影主角與隱知墨試圖取走古鏡時，靈骨作為試煉的一部分在秘室中現身。其存在與修行者的靈魂磨練及隨後出現的心魔考驗密切相關。
