# 吉娜氣魄

吉娜的精魂型態，受馬茲元精掌控。

## 背景資訊

吉娜的精魂型態。在《鬼城懺七》章節的「地宮元神」任務中登場。馬茲席塔手中握有她的精魂氣魄，並以此為要脅，使吉娜不得不作為內應待在馬茲元精身邊。此型態具備「精魂氣魄」的特徵，被擊敗後會變為反色靈體並重生，唯有使用「血刃」方可將其徹底消滅。
