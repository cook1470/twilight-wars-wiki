# 哈斯卡大帝

哈斯卡帝國的創立者，皇家騎士團的最高領袖，繼承了不老不死的皇家血脈。

## 背景資訊

哈斯卡帝國的建立者與統治者。

在能源浩劫與污染風暴摧毀文明後，哈斯卡大帝建立了帝國，帶領人類重登物種頂端。他致力於尋找並公平分配稀有的日光資源，其使命是恢復地球往日的風采。

他擁有不老不死的皇家血脈，並育有一對雙生兒女：阿波羅與雅堤米絲。作為帝國的權力核心，他以強大的意志力維護著社會的秩序，認為日光是全地球的資產，應由強大的集權力量守護。
