# 隱心氣魄

隱心的靈魂型態，受馬茲元精掌控。

## 背景資訊

隱心的精魂型態。在《忘情谷》中，受馬茲席塔元精的攝心術影響而攻擊玩家。在此狀態下被擊敗後會變為反色靈體，一段時間後會重生，需使用具備「血刃」能力的武器方可徹底消滅。
