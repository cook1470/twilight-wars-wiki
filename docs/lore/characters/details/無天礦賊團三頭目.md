# 無天礦賊團三頭目

無天礦賊團幹部，帶領成員在邊境地殼裂縫挖寶。

## 背景資訊

無天礦賊團的三號幹部。

在任務「穿越北方邊境」中出現。率領成員在北方地殼裂縫一帶伺機挖掘稀有礦物。主要使用大刀與霰彈槍。
