# 隱奘 (影俠)

天隱門弟子，後加入天影十字軍並失蹤。

## 背景資訊

隱奘原為天隱門弟子，師從眉隱大師。後加入天影十字軍並獲得「影俠」稱號。在噬魂秘藥事件後失蹤，與馬茲席塔的木舍利結合成為七月教教主，最後透過斷魂術轉生至裏世界。
