# 隱奘 (影俠)

天影武士精英，負責調查紅河屍毒。

## 背景資訊

天影十字軍的高階成員。

在「紅河古墓」與後續任務中，與玩家並肩作戰，調查屍毒的起源並尋找血清。他展現了極強的戰鬥力與決斷力，是天影議會在該地區的核心執行官。
