# 多倫大公

地球復原組織高級科學家，皇家大公。

## 背景資訊

### 身分與地位
多倫大公 是哈斯卡帝國的高級科學家，隸屬於皇家獨立行政單位「地球復原組織 (Earth Blue Organization)」。他同時具備大公的貴族爵位，是帝國科研與行政體系中的重要人物。
