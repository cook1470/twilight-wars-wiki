# 伊芙

伊拉馬族的少女，長老的親族。

## 背景資訊

伊拉馬族的少女。在《迷霧森林》任務中被盜礦賊囚禁，後被天影武士救出。隨後在《天火的試練》與《火焰晶石的秘密》中引導玩家進入伊拉馬族的聖地。

### 語言特徵

伊拉馬族人平日使用簡單的日文羅馬拼音通訊。在與玩家（天影武士）互動時，則會使用對方語言的單字拼湊成破碎的句子來表達意圖（如："救人. 天影. 感謝."）。
