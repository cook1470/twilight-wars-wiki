# 伊芙

伊拉馬部落的成員，在「迷霧森林」中由天影武士救出並護送。

## 背景資訊

伊拉馬部落的成員。

在任務「迷霧森林」中最初以「伊拉馬人質」身份出現，被困於地圖東南方的盜賊營地。當玩家擊敗北方的部落戰士並獲得長老贈與的「水晶弓」前來時，她才會揭露自己的真實姓名為「伊芙」，並跟隨玩家行動。

她不具備戰鬥能力，也不會參與任何作戰。在任務中，玩家必須在盜賊首領回巢的威脅下，保護並引導她平安回到北方長老處。
