# 皇家騎士團 人物誌

## 具名角色 / 核心英雄

- [**亞拜爾上校**](<./details/亞拜爾上校.md>) - 駐守於北方邊境地殼裂縫區域的帝國軍官，率領境衛軍。
- [**克萊德騎士**](<./details/克萊德騎士.md>) - 皇家騎士，出現在《尼斯水宮》任務。
- [**哈斯卡大帝**](<./details/哈斯卡大帝.md>) - 哈斯卡帝國的建立者，繼承不老不死的皇家血脈。
- [**多倫大公**](<./details/多倫大公.md>) - 地球復原組織高級科學家，皇家大公。
- [**帝國巡邏長**](<./details/帝國巡邏長.md>) - (待補充)
- [**帝國隨扈**](<./details/帝國隨扈.md>) - (待補充)
- [**查理**](<./details/查理.md>) - 皇家科學院研究員，駐守於阿薩斯城，協助進行上古冰劍的能量測試。
- [**特戰隊員**](<./details/特戰隊員.md>) - (待補充)
- [**皇家量產機械部隊**](<./details/皇家量產機械部隊.md>) - (待補充)
- [**皇家騎士**](<./details/皇家騎士.md>) - (待補充)
- [**阿波羅**](<./details/阿波羅.md>) - 哈斯卡大帝之子，皇家第一騎士，統帥皇家騎士團的領袖，與九方一色是生死之交。
- [**阿波羅元帥**](<./details/阿波羅元帥.md>) - (待補充)
- [**阿留申**](<./details/阿留申.md>) - 皇家騎士團天位騎士，在戰場上會主動尋找武器進行戰鬥。
- [**阿薩斯**](<./details/阿薩斯.md>) - 阿薩斯城的領袖，暗中瞞著帝國進行上古「冰劍」的祕密研究。
- [**阿薩斯士兵**](<./details/阿薩斯士兵.md>) - 阿薩斯城的基層士兵職位。
- [**阿薩斯守衛**](<./details/阿薩斯守衛.md>) - 駐守於阿薩斯城的守衛職位。
- [**阿薩斯小隊長**](<./details/阿薩斯小隊長.md>) - (待補充)
- [**阿薩斯研究員**](<./details/阿薩斯研究員.md>) - 隸屬於阿薩斯研究團隊的科學人員。
- [**阿薩斯騎士**](<./details/阿薩斯騎士.md>) - 阿薩斯城的騎士職位。
- [**阿薩斯騎士副團長**](<./details/阿薩斯騎士副團長.md>) - (待補充)
- [**阿薩斯騎士團長**](<./details/阿薩斯騎士團長.md>) - 阿薩斯城騎士團的指揮官職稱。
- [**雅堤米絲**](<./details/雅堤米絲.md>) - 哈斯卡大帝之長女，靈能研究所主席，創造了「光印」技能。
- [**靈能訓練官**](<./details/靈能訓練官.md>) - (待補充)
- [**魏斯蒙少尉**](<./details/魏斯蒙少尉.md>) - 皇家騎士團少尉，新生訓練營教官。
