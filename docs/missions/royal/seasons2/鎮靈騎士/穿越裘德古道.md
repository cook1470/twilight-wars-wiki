# 穿越裘德古道

[回到章節：鎮靈騎士](<../鎮靈騎士.md>)

你必須和阿留申(Aleutian)儘速穿越裘德古道，前往索爾之門與其他天位騎士們會合。

- **失敗條件**：阿留申死亡
