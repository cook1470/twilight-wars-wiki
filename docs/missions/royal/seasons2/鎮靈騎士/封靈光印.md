# 封靈光印

[回到章節：鎮靈騎士](<../鎮靈騎士.md>)

一般對靈體的認識是戰死的士兵體內升華出來的異次元生命體，擁有極高智慧與不同形態的戰鬥方式。

靈體生命和實體生物可以有一定程度上的互動，但無法直接奪取彼此的生命。

皇家的秘技「光印」由皇女雅堤米絲所創，是一項帶有魔導色彩的近武技能，進入光印的生物會瞬間移轉至時空隧道，再被彈出至原屬次元。

所謂的靈體封印，其實是使用光印，將靈體傳送回其所屬次元。

你必須在靈能研究所的光印操練室熟練這個技能才能繼續接下來的任務。

- **過關條件**：封印魔造靈體
- **失敗條件**：阿留申死亡
- **任務地圖**：待補充
