# 赤焰魔山

[回到章節：赤焰王女](<../赤焰王女.md>)

你與阿波羅先後到達赤焰山登山口，然後一同向上爬去。

山勢險峻，好在你們身手不凡，一路上還能談談說說，阿波羅為人隨和，便簡略道出數十年前他與九方一色的一番恩怨。

原來九方一色少年時候在皇家騎士團中，與阿波羅曾是對一同出生入死的換帖兄弟，彼此承諾未來無論何事都願捨命相助。九方一色後來離開皇家，自己開創一片天地，在中央大陸北方的寒地建立了國家，成為一國君主。

幾經寒暑，九方婚後有了一女，那便是九方織夢了。九方的妻子早逝，因此對這獨生女極其寵愛，但因北境國事繁忙，卻對女兒少了管教。

九方織夢成年後出落得花兒一般美麗，一頭烏黑秀髮襯得她嫩臉更加白皙，追求的少年自是多不勝數，但她除父親外，對所有男子一概冷顏以對，因而得了個「冰玉王女」的稱號。誰知那年九方織夢竟離家出走，一去音訊全無。

九方一色放下國務，摘下皇冠，背離子民，就為了找出女兒的下落。十年過去，九方終於在傭兵團找到了女兒的下落，但那時聽說九方織夢已是不治之身。

原來當年九方織夢跟著一個外地青年加入傭兵團闖蕩江湖，兩人攜手共進，青年打敗強敵成為傭兵團五手，九方織夢也順利站上傭兵團的六手。兩人為求更進一步，各自開始了秘密修練，沒想到最後九方織夢卻疑似走火入魔，倒在無人能進的赤焰山中。

那青年五手為了九方織夢進入了赤焰山，卻遇上同來追尋女兒的九方一色。兩人展開一場昏天地暗的戰鬥，最後青年五手負傷離去，九方為了保全女兒的成就，便接下了傭兵團五手的位置。

九方後來找過阿波羅兩次，但阿波羅已承諾姊姊雅堤米絲，絕不對人類使用聖光復生術。兩人為此大打出手，且越打越烈，直到最後訂下了約定，九方方才罷手。這約定便是未來若能強押阿波羅到九方織夢的身前，阿波羅就需以復生術救治九方織夢。

你聽著阿波羅訴說往事，心中卻想，這次深入赤焰山，想必九方織夢就在其中。雖然尚未開打，卻早已輸給了九方一色...

- **過關條件**：助阿波羅平定靈體之亂
- **失敗條件**：阿留申死亡
