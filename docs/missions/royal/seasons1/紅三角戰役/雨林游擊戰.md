# 雨林游擊戰

[回到章節：紅三角戰役](<../紅三角戰役.md>)

沿著紅河進入茂密的雨林，在陰暗的森林裡，你看見倏忽而過的天影武士。

你不清楚天影在此的目的為何，但身為皇家騎士團的一員，不能坐視他們在自然保護區內恣意橫行。

拿出你的本事，在短時間內給予天影重創，讓他們知難而退。

- **過關條件**：五分鐘內擊敗20位以上的敵人
