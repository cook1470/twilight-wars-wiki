# 洗淨罪惡

[回到章節：紅三角戰役](<../royal/seasons1/紅三角戰役.md>)

戰事來得奇，去得快，傭兵團與天影的戰力已退出大半，帝國也派出後援部隊，準備再次啟動紅河三角洲的防衛工作。

在這場戰役中升為小隊長的你，卻在此時發現一批傭兵團的人馬從雨林中奔出。

不能放過這些可惡的生態破壞者，帶著你的小隊完成這場戰役最後的工作吧！

- **過關條件**：時間內擊滅欲逃脫的傭兵
- **失敗條件**：任一傭兵逃出戰場
- **任務地圖**：待補充
