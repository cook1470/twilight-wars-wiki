# 希瓦暗道

[回到章節：盜體合靈](<../third/seasons2/盜體合靈.md>)

據青嵩散人所言，元體舍利藏在希瓦邊城的某間秘室，而進入邊城的關口都有重兵駐守，一般極難進入。

不過青嵩散人有吉娜這位極為厲害的弟子在七手身邊作內應，得知尚有一條隱密通道可從地底進入元體舍利所在的秘室，但只知元精曾數次由此暗道進出，內中有無風險則不得而知。

來到暗道的入口，聞到一股腐泥敗味飄散而出，你必須深入暗道，並找出藏有元體舍利的秘室所在。

- **過關條件**：尋找元體所在
