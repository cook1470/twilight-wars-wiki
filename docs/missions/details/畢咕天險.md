# 畢咕天險

[回到章節：赤焰王女](<../royal/seasons2/赤焰王女.md>)

擊退入口大廳的黑炎部隊後，少數熟悉尼斯水宮的天位騎士便深入水道深處探索靈能者消失之秘，其餘人馬仍在宮外待援對抗黑炎大軍。

阿波羅快速地安排了各人行止後，便邀你一同前往九方一色在信中所提到的赤焰山。

赤焰山是座低矮的活火山，周遭無人居住，噴發活動頻繁的山口外圍繞著充滿兇猛野獸的險惡灌木林，讓所有前來探險的獵戶與旅人卻步。

你與阿波羅來到赤焰山的附近，卻在林中走散了。幸好事先已約定了會合地點。

你朝著登山口前進，卻被一條寬闊的河水擋住，河面湍急洶湧，水中似乎也暗藏魔物，但是為了儘快與阿波羅會合救出雅堤米絲，你沒有時間繞道尋路，只能緊握拳頭踏水而進。

- **過關條件**：穿越裘德古道
- **失敗條件**：阿留申死亡
- **任務地圖**：待補充
