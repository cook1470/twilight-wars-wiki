# 登上風月寶塔

[回到章節：風月寶鏡](<../skydow/seasons2/風月寶鏡.md>)

雖然風月寶鏡之說在天隱一族之中已流傳百年，但自皇家進駐丹格斯後，誰也沒再見過此一古鏡。

傳說寶鏡放在塔頂某處吸納月光精華，你必須登上寶塔頂層，和隱知墨一同設法尋找藏鏡所在。

- **過關條件**：登上寶塔頂層
- **失敗條件**：隱知墨死亡
