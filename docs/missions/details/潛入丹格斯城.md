# 潛入丹格斯城

[回到章節：風月寶鏡](<../skydow/seasons2/風月寶鏡.md>)

隱知墨與你守在丹格斯城外數日，已摸清城內守衛的巡迴規律。保管風月寶塔大門鑰匙的守衛隊長，每日正午會帶一小隊進塔換班。

依計畫，你必須等到守衛隊長帶隊進塔後，設法接近寶塔大門，然後在下哨隊伍出塔後，立即從守衛隊長身上奪取鑰匙以進入風月寶塔。

- **過關條件**：打開風月寶塔之門
- **失敗條件**：隱知墨死亡

城內居民被誤殺致死
