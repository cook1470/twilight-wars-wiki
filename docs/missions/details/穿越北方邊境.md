# 穿越北方邊境

[回到章節：地火靈魂的傳說](<../skydow/seasons1/地火靈魂的傳說.md>)

位於艾爾瑪北方的帝國邊境是個大型的地殼裂縫，帝國軍常年在此駐守。

地殼裂縫中偶能發現稀有礦物，因此也有許多淘金族及盜賊冒死來此挖寶。

為了不讓帝國發現你對邊境外的世界有興趣，你必須避開帝國軍的耳目，悄悄地穿越邊境。

- **過關條件**：穿越北方邊境
- **失敗條件**：任一帝國士兵發現天影的蹤跡
- **任務地圖**：待補充
