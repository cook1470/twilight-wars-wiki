# 跟入敵窟

[回到章節：艾爾瑪的紛亂](<../skydow/seasons1/艾爾瑪的紛亂.md>)

上一場戰鬥結束時，你發現一名盜賊躲藏在戰場邊緣，並準備逃走。你悄無聲息地繞到他的後方，打算跟蹤他查看盜賊總部的入口。

通往盜賊總部的路上想必也有許多敵人把守著關口，你必須小心，不要讓逃跑的盜賊發現你的跟蹤。

- **過關條件**：跟蹤逃跑的盜賊進入賊窟
- **失敗條件**：逃跑的盜賊離開視線或死亡
- **任務地圖**：待補充
