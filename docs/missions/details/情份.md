# 情份

[回到章節：綁架阿曼達博士](<../third/seasons1/綁架阿曼達博士.md>)

得知傭兵團即將對伊蒂絲不利的消息，不如往常地有種異樣的感覺浮上心頭。

你覺得在理清和伊蒂絲之間的牽絆之前，不能讓她就這麼死去。

你決定暗中將伊蒂絲從傭兵團的手中救出。

- **過關條件**：將伊蒂絲救出
- **失敗條件**：伊蒂絲死亡
- **任務地圖**：待補充

::: info 開啟條件
在結尾不要拾取吉娜給的寶箱，直接往下走，可以獲悉傭兵團的陰謀，開啟隱藏任務『情份』。
:::
