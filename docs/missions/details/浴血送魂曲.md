# 浴血送魂曲

[回到章節：斷魂殘影](<../third/seasons2/斷魂殘影.md>)

靈核之主在吸收精核之後，距離三核合一僅剩最後一步。此時，馬茲希塔決定再次施行斷魂之儀。

不同於古老流傳的方法，馬茲希塔依據自己兩度以靈體徘徊於生死邊界時所領悟的六道奧義，創造出一種全新的儀式——專門用來將隱煞的意志從其肉身中剝離的「裂魂之儀」。

這個儀式的目的，是切斷木之聖遺與隱煞之間的血脈連結。如此一來，靈核之主便能吸收木之聖遺，重塑自身肉體，而隱煞也可能藉此尋回失落的記憶。

然而，一旦儀式開始，真正會發生什麼，無人能夠預料。對馬茲希塔與隱煞而言，這是一場沒有退路的賭局，其結果亦無人能夠預知。

隱心在擺脫縛心之術後，得知了馬茲希塔的計畫，並選擇出手相助。畢竟，隱煞的命運始終是她心中最為牽掛之事。

- **過關條件**：完成斷魂術
- **失敗條件**：隱心或吉娜死亡
- **任務地圖**：待補充
