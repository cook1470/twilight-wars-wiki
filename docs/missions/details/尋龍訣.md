# 尋龍訣

[回到章節：龍魅奇謀](<../skydow/seasons2/龍魅奇謀.md>)

與隱知墨走失，被捲入奇異的「腦洞」旋渦，又遇上這來路不明的腦洞之主，一切都發生的不太真實又令人費解，只好強迫接受腦洞主的一翻說詞。

「我舊姓蕭，悟道後已不著姓名，你就稱呼我為蕭居士吧。每顆人腦的正中都有一個空洞，由空氣所撐，人死後，氣散洞消，因為無從觀察研究，所以各醫家學者皆不識腦洞之謎，認腦洞之學為無稽之談。然而這腦洞確實存在，並連接著人所不知的一片新天地，一個更為真實存在的世界。

腦洞連接了兩個世界，所謂『悟道』之道，指得便是這穿梭兩界的通道。雖然生靈有許多來去兩界的方法，但唯有通過腦洞才能連結兩界的記憶與人格。表界中少數悟道之人，甚至能在腦洞裏創造出延伸自裏世界的空間，並在其中修練靈劍...唔...雖然也有尚未悟道、天生便持有靈劍的特例。

我需要你和我一起走入異界，找尋另外一人的腦洞。十年前我在那人的腦洞之中施展靈體分離之術，元神蟄伏在那人腦洞，元體在表界牽引因緣，而我元靈則在鏡中等待。如今元體已引得表界一陣波瀾，而你又在此時進入我的腦洞，一切的機緣巧合，套句表界術語，正是你的天影原力引領著你來到此處。

幫助我破除那人腦洞中的封印。走吧！這一趟不只救人，也是你必經的天影之道！」
