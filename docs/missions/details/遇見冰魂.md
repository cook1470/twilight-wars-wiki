# 遇見冰魂

[回到章節：冰劍檔案](<../royal/seasons1/冰劍檔案.md>)

南極冰原在大災難後成為一個幾乎沒有生命的區域，浮在南極海面的巨大冰蓋和深海的地底似乎有了某種聯結，冰原上有許多地方終年冒著致死的冰凍氣體。

但比起這片冰天雪地，更另人感到寒冷的，是不時出現在風雪中一雙雙的青藍瞳眼。
