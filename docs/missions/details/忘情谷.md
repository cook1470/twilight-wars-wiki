# 忘情谷

[回到章節：斷魂殘影](<../third/seasons2/斷魂殘影.md>)

隱心帶著一具你無法辨認其真實身分的身軀——究竟是隱煞，還是靈核之主——與你一同離開赤日祭壇，向南而行。

沿途，隱心向你說明了第七月教團的由來。這個教團是由馬茲希塔所建立，他藉由掌控隱煞的身軀，以其不死之身與強大的天影氣息吸引了眾多高手投靠，其中甚至包括許多天影一脈的強者。然而，身為教主的隱煞似乎對自己的過去毫無所知，對馬茲希塔所設下的諸多限制也毫不在意。於是，教團內逐漸分裂為兩個派系：一為忠於天影血脈的內壇一脈，另一則是以傭兵為主、實際聽命於馬茲希塔的外壇勢力。

隱心回憶道，她與隱煞自幼便是青梅竹馬。隱煞失蹤後，她再也無法回到原本的生活，於是踏上尋找他的旅程。當第七月教團現世時，她立刻前往確認。儘管外貌與氣質有所不同，她仍在第一眼便認出教主正是隱煞。為了幫助隱煞尋回失去的記憶，她以侍從的身分加入教團。

然而，隨著時日推移，隱心卻驚訝地發現，隱煞的眼中沒有絲毫迷惘或失憶的痕跡。他深信自己乃天命所歸之人，等待著某個能引出他體內無限增生之力的契機，並相信藉此能進入虛無之境，窺見宇宙的真理。

如今，隱煞與靈核之主皆已無法言語，你帶領眾人前往魔窟山脈中青松散人的隱居之所，希望這位故人能為你們指引前行的方向。

- **過關條件**：找出喚醒元靈師父之法
- **失敗條件**：隱心死亡
- **任務地圖**：待補充
