# 花落水流

[回到章節：黑色追殺令](<../third/seasons1/黑色追殺令.md>)

你與神秘隱居士一同前往豐州大陸的銀杏林，尋找吉娜的蹤跡。

- **過關條件**：與吉娜交涉
- **失敗條件**：吉娜或隱居士死亡
