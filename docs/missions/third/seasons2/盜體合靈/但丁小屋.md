# 但丁小屋

[回到章節：盜體合靈](<../盜體合靈.md>)

但丁小屋地處偏僻，是遠古詩人住所的遺址，在舊世界曾是對抗星幣文化的重要據點，如今卻是元靈師父與你躲避七手追捕的避難地。

元神之間互有感應，因此只要與元靈師父在一起，就無法在一處躲過七手的耳目超過三個月。

這一次待在但丁小屋已有半年之久，原以為此處可能有特殊地氣，讓元精七手無從感知，卻沒料到元精七手其實已經察覺，暗中集結眾多傭兵，想要趁你不備之際一舉攻入。

大批七手爪牙的攻入讓你措手不及，但跟隨元靈師父的這些年，已從師父手上習得一身出神入化的功夫，此時便是你使出渾身解數，對抗惡勢力的時候了。
