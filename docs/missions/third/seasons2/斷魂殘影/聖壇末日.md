# 聖壇末日

[回到章節：斷魂殘影](<../斷魂殘影.md>)

你來到第七月教團的赤日祭壇，發現四周被傭兵門徒團團包圍——他們正是馬茲希塔昔日的部下。顯然這些人是奉第七手精核之命駐守於此，一方面守護祭壇，另一方面暗中監視教主的動向。
