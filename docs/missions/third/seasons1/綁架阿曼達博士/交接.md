# 交接

[回到章節：綁架阿曼達博士](<../綁架阿曼達博士.md>)

傭兵團指定交人的地點在維克多峽谷的盡頭，也就是俗稱的死亡谷。死亡谷的地勢就像被斧頭鑿過一樣，有著許多深溝。

橫越斧頭幫的地盤，將伊蒂絲交接給傭兵團的使者就算完成任務。

你是可以就這麼簡單地完成任務！

- **過關條件**：將伊蒂絲安全交給傭兵團
- **失敗條件**：伊蒂絲死亡
- **任務地圖**：待補充
