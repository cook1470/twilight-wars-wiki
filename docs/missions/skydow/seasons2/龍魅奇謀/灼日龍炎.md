# 灼日龍炎

[回到章節：龍魅奇謀](<../龍魅奇謀.md>)

你即將進入的腦洞，據蕭居士所言，乃表界一名女子高人所想。她也是位靈劍使，在腦洞中練功時不慎走岔了真氣。在她心神大亂而腦洞大開之際，巧得異界靈佛之助，慢慢導火趨魔，將那走岔的真氣散入異界，而這股屬性特殊的真氣，不幸從異界深處引來了傳說中的巨大魔物。

那名女子為了不讓魔物來到表界，將蕭居士也召了進來，可惜魔物太過強大，合二人之力仍抵不住，女子心一橫，就要自盡毀壞其身，讓魔物不得其門而出，但那異界靈佛搶先一步制住了她元神，鎖了她腦洞，不但阻了魔物出世也算救了那名女子。但因元神被封在腦洞，女子陷入了十年長眠。

那女子的精神力異常強大，雖然魔物被關後怒不可抑，在腦洞興風作浪，但她十年間還是以強大念力慢慢將魔物侷限在腦洞一隅，但人不比萬年魔物，能僵持多久，誰也說不準。如今她腦中亂象已成，即使破除靈佛的結界，女子仍舊深陷與魔物的角力，想要救人，只能硬碰硬，設法滅了那魔物。聽了這番始末，你心中認定這名女子值得一救，決意隨蕭居士進她腦洞中的龍穴放手一搏。
