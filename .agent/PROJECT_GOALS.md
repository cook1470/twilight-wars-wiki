# 🎯 專案目標：光暈戰記 Wiki (Neo Twilight Wiki)

本文件定義了新一代《光暈戰記》Wiki 的核心願景與長期開發目標。

## 🌟 核心願景
建立一個純淨、專業、且 AI 友善的《光暈戰記》百科全書，取代資訊混亂的舊版 Wiki，為玩家與開發者提供最權威的資料參考。

## 🚩 長期目標
1.  **資料全面遷移**：將 Fandom Wiki 與巴哈 Wiki 的正確資訊（陣營、武器、技能、地圖）完整遷移至此。
2.  **極簡視覺體驗**：採用 VitePress 驅動，提供無廣告、秒開的閱讀體驗。
3.  **結構化數據**：為每一件武器與技能建立結構化的 Markdown 頁面，方便 AI 檢索。
4.  **社群貢獻**：未來開放透過 GitHub PR 進行資料補充。
